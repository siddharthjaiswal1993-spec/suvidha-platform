# Official Sources

Bibliography for `docs/INDIA_DIGITAL_SERVICE_LANDSCAPE.md` and `docs/INTEGRATIONS.md`. Organised by
the same category headings as the landscape document. All sources were located and reviewed via
web search in **July 2026**; "accessed" dates below all mean that month unless stated otherwise.
Official `.gov.in` / regulator-domain sources are preferred and marked **(official)**; where the
only available corroboration was a secondary/trade-press source, it is marked **(secondary)** and
the corresponding landscape-doc confidence rating reflects that.

## Notes on sourcing methodology

- Every source above was located via targeted web search conducted in July 2026 and cross-checked
  against at least one other result in the same search batch where possible.
- Where only secondary/trade-press sources were found for a fact that materially affects Suvidha's
  product claims (e.g. the PPF/ECS death-claim exclusion, the professional/trade-licence landscape),
  the corresponding entry in `docs/INDIA_DIGITAL_SERVICE_LANDSCAPE.md` is explicitly rated **medium**
  or **low** confidence rather than presented as settled fact.
- No specific API endpoint URL, request/response shape, or rate limit is asserted anywhere in this
  documentation set unless it was directly found in an official specification (the DigiLocker
  Issuer/Requester API being the only case in this landscape where such a specification is genuinely
  public) — see `docs/INTEGRATIONS.md` for how that distinction is carried into the mock-adapter design.
- This bibliography should be treated as a living document: several of the systems described above
  (Bima Sugam, DPDP Rules phased commencement, UIDAI-RGI death-data sharing) are actively evolving in
  2026, and re-verification before any real production decision is strongly recommended.
